# Computational Psychiatry Starter Kit

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20559868.svg)](https://doi.org/10.5281/zenodo.20559868)

A ready-to-use template for computational psychiatry research — Docker environment, example notebooks, and sample data.

Part of the [comp-psych](https://github.com/comp-psych) initiative.

## Quick Start

```bash
docker-compose up
```

Then open `http://localhost:8888` in your browser to access the Jupyter notebooks.

## Project Structure
